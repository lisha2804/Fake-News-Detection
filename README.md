# Fake-News-Detection

This project focuses on detecting whether a news article is *Fake* or *Real* using machine learning techniques.

---

# Project Objective

With the rapid spread of misinformation online, fake news detection has become crucial.  
The goal of this project is to build a machine learning model that can automatically classify news articles based on their content.

---

## Machine Learning Algorithms Used

The following classifiers are implemented and evaluated:

- Logistic Regression  
- Decision Tree Classifier  
- Random Forest Classifier  
- Gradient Boosting Classifier  

---

## Technologies Used

- Python  
- Jupyter Notebook  
- Pandas  
- NumPy  
- Scikit-learn  
- Natural Language Processing (NLP)
